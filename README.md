# Inventory Management System

A simple yet useful C program  to take care of inventories on a small vegetable shop.

```NOTE: The password 'gaida' is used throughout the application.```

# Features:
   - Basic CRUD operations on a local file.
   - Search a particular item to reveal its quantity, rate and cost and calculate profit/loss on that specific item.
   - View complete details of items, both sold and bought and calculate overall profit/loss.
   - Password protection while trying to perform CRUD operations as well as while viewing items.
